# x86 Software Cleanup Checklist

## Purpose

This checklist summarizes Intel (x86) software found on your Apple Silicon Mac and provides practical cleanup suggestions.

Use this as a safe reference before uninstalling anything.

---

## 1) Root cause already identified (important)

- Your `xcode-select` was pointing to an old Xcode in `~/Downloads/Xcode.app` (15.2), while `/Applications/Xcode.app` is newer (26.4).
- This caused `brew install node` to fail.

### Fix command

```bash
sudo xcode-select -s /Applications/Xcode.app/Contents/Developer
xcode-select -p
xcodebuild -version
```

Expected: path under `/Applications/Xcode.app/...` and Xcode 26.x.
